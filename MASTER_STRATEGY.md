## Unified Business Strategy: Student + Professional Tiers

**Last Updated:** December 2025
**Status:** MVP Complete, Scaling Phase

## Executive Summary

ScholarSync is the only all-in-one ethical AI toolkit serving two distinct markets:

1. **Student Tier (B2C)** - Budget-conscious students needing AI disclosure, prompt coaching, and research tools
2. **Professional Tier (B2B)** - SMBs and enterprises needing AI compliance and workforce training

**Core Differentiator:** Prompt Coach with 0-100 scoring - no competitor offers student-focused prompt education.

**Revenue Model:** Freemium with tiered subscriptions across both markets.

## Mission

Help students and professionals use AI responsibly, transparently, and effectively - not to cheat or cut corners, but to learn and work better.

## Core Features

### 1. AI Disclosure Statement Generator

**What:** Auto-generate professional, citation-style AI disclosure statements.

**Output Formats:** APA, MLA, Chicago, Generic, IEEE, Harvard

**Key Differentiators:**
- Works with ANY AI tool (not locked to ecosystem)
- Detailed contribution breakdown
- Version history
- Course-specific templates

### 2. AI Prompt Coach (PRIMARY DIFFERENTIATOR)

**What:** Score prompts 0-100 with actionable improvement suggestions.

**Scoring Rubric:**
- Clarity (0-25): Is the request understandable?
- Specificity (0-25): How specific is the task?
- Context (0-25): Is background provided?
- Structure (0-25): Is it well-organized?

**Educational Features:**
- Real-time feedback as you type
- Before/after comparison
- Learning tips based on score
- Target AI selector (ChatGPT, Claude, Gemini)

### 3. Academic Integrity Toolkit

**Components:**
- Plagiarism Detection (premium)
- Citation Verification (planned)
- Source Finder with real academic papers
- Writing Style Analyzer

### 4. Research Assistant

**What:** Search 200M+ academic papers with AI-powered relevance ranking.

**APIs:** Semantic Scholar, OpenAlex (FREE)

**Features:**
- Auto-generate citations (APA, MLA, Chicago, BibTeX)
- Save to personal library with folders/tags
- Abstract summarization
- Related papers discovery

### 5. Grammar & Writing Check

**What:** LanguageTool integration for grammar, spelling, style.

**Features:**
- Inline error highlighting
- Suggestion cards with accept/ignore
- Readability scoring
- Academic tone checker

### 6. Assignment Audit (NEW - December 2025)

**What:** Pre-submission review that gives students a brutally honest third-party perspective on their work before submitting.

**CIDI Framework Scoring (0-100 each):**
- **Context** (20%): Does the work establish sufficient background?
- **Integrity** (30%): Is the argument logically sound and consistent?
- **Details** (25%): Are specifics provided rather than vague generalities?
- **Insight** (25%): Does the work show original thinking?

**Key Features:**
- Third-party summary ("What a reader would take away")
- Requirements checklist (if rubric provided)
- Gap analysis with priority ranking and fix time estimates
- Improvement suggestions with effort/impact ratings
- Issue detection (repetition, vague statements, unsupported claims)
- Re-audit comparison to track improvement
- Text statistics (word count, reading level, citations)

**Evidence-Based Analysis:**
- NO hallucinations - only quotes from actual text
- Every claim backed by specific location references
- Actionable feedback with examples
- Brutal honesty over false encouragement

**Tier Placement:**
| Feature | Free | Student ($4.99) | Pro ($7.99) |
|---------|------|-----------------|-------------|
| Audits/month | 1 | 10 | Unlimited |
| Gap analysis | Basic | Full | Full |
| Custom rubric | No | Yes | Yes |
| Re-audit | No | 3x | Unlimited |
| Requirements matching | No | Yes | Yes |

## Technical Architecture

### Stack

- **Frontend:** Next.js 14, React, TypeScript, Tailwind CSS, shadcn/ui
- **Backend:** Next.js API Routes, Supabase Edge Functions
- **Database:** Supabase (PostgreSQL) with RLS
- **Auth:** Supabase Auth (Email + Google + SSO for enterprise)
- **Payments:** Stripe (subscriptions, usage-based)
- **AI:** Groq (fast inference), Claude API (complex tasks)
- **Hosting:** Cloudflare Pages (zero-cost until scale)
- **Mobile:** React Native + Expo
- **Extension:** Chrome Manifest V3

### Cost Structure

| Service | Cost | Notes |
